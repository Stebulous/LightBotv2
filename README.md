## Installation

Clone:
```sh
$ git clone https://github.com/Stebulous/LightBotv2.git
```
Then install the package:
```sh
$ npm install
```
Now you can run it:
```sh
$ npm run bot
```
## To manage with ``pm2``  
install [pm2](https://github.com/Unitech/pm2)  

```sh
$ cd PATH/TO/LightBotv2/lib
$ pm2 start bot.js --name="NameForTheProcess"
```
